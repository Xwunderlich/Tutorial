Tutorial: Intro to JavaScript

What is JavaScript?

  1. JavaScript is a prototype-based scripting language developed by Netscape Communications in 1995
  2. Despite its name, JavaScript is derived from C rather than Java
  3. Commonly seen embedded in HTML for dynamic web pages, but also increasingly seen in server-side applications

JavaScript basics

  • JavaScript can either be within HTML <script> tags or in a separate file linked from the header
  • JavaScript embedded in this way is interpreted rather than compiled
  • Typing is dynamic and as such JavaScript does not support method overloading. It will only use the most recent definition of a method.
  • There are no classes in JavaScript, although it is object-based
  • Scoping is function-level

JavaScript syntax

  1. Objects
    objects are defined as follows with a colon between the attribute and the value 
    var class = {type:"CSE", number:"3901", name:"Web Apps"};
  2. Functions
    functions make up most of JavaScript and are defined as follows: 
    function add(a,b){
  return a+b;
}

    Note: functions do not require a return statement
  3. Listeners
    4. document.getElementById("id").addEventListener("click",function);

    there are various other events that can be listened for including: keyup, change and keypress.

DOM (Document Object Model)

The DOM is the interface between JavaScript and the HTML tree.

Accessing the DOM

The DOM is accessed by using a get method though the document object 
document.getElementById("identification");
//this get method retrieves a DOM element based on Id
document.getElementsByTagName("a");
//this get method retrieves DOM elements based on their HTML tag name
//NOTE: this method returns an array of elements
document.getElementsByClassName("subclass");
//this get method retrieves DOM elements based on their class
//NOTE: this method returns an array of elements

Properties of DOM elements

element.innerHTML= //insert desired contents of the element here 
element.attribute= //insert desired value of the element's specified attribute here 
element.style.property= //insert desired value of the element's CSS style property here
