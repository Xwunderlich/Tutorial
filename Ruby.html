What is Ruby?

  1. Ruby is an object oriented general purpose programming language.
  2. Developed in the mid-1990s by Yukihiro Matsumoto.
  3. Ruby is a coss platform language.
  4. Popular with rails for serverside programming.

Why learn Ruby?

It is a unique language designed with the programmer in mind.

Ruby basics

  • Syntax is somewhat flexible, but the prevailing wisdom is to use very turse syntax.
  • Ruby code is interpreted, not compiled.
  • Ruby is dynamicly typed.
  • Ruby is object oriented.
  • Ruby has classes.
  • Memory Managment is automatic.

Getting started

You can see how to install Ruby, and run interactive ruby (IRB) on your machines. However, for the purposes of this tutorial, and to see how you like Ruby, I would reccomend following along in this online IRB website.

Hello world!

Lets get started with the basics.


>> puts "Hello World"

Will result in this output in IRB:


Hello World
=>nil

Simple Function

Lets apply this to a simple function.


>>  def foo
      puts "Hello World"
    end
>>  foo

Will result in this output in IRB:


Hello World
=>nil

Lets break down the code above. Def is the signal that we are defining a function. End tells the interpreter where the definition ends, everything inbetween is part of the function named foo. We can then call foo to execute the code within the function. This may be syntacticly differnt from languages which use nested brackets, but logically it is similar.

If then statement

Lets look at an If, Then statement in Ruby.


>>  x=1
    y=1
    if (x==y)
      puts "X is equal to Y."
    else
      puts "X is not equal to Y."
    end

Traditionally in Ruby the parentheses are ommited, as below.


>>  x=1
    y=1
    if x==y
      puts "X is equal to Y."
    else
      puts "X is not equal to Y."
    end

Which should return:


X is equal to Y.
=>nil

Conditional & logical opporators should be intuitive if you have programed before. You can always look up any Ruby operatos you are unsure of.


Do it yourself!

Try to design a function which can determine if a given number is positive or negative. You can test your function by passing differnt values. Be sure to test boundry cases, and verify that your function works as expected.


Loops in Ruby.

Loops should be familiar to people with a little programming experience, but there are also a couple that are not so common.


  • while
  • for
  • until
  • break
  • redo
  • retry
>>  while i < 100
      i+=1
    end

This will increment i up to 100. Note that there is no ++ incrementor in Ruby.


Arrays in Ruby

Arrays in Ruby can have mixed types, including arrays. Negative indicies represent a location relative to the end of the array. Indicies start at 0. Arrays can be created in a couple ways.


Literal

ray [3, a, "Robbie", 3.1415]

=> [3, a, "Robbie", 3.1415]

Array.new

ray = Array.new []

=> []

Array.new with length

ray = Array.new(4) []

=> [nil, nil, nil, nil]
